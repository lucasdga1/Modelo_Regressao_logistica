# Modelo_Regressao_logistica
## Este repositório é dedicado a um modelo de regressão logística usado para prever a propensão a doenças cardiovasculares

* Objetivo: Criar um modelo de regressão logística capaz de prever a propensão a doenças cardiovasculares
* Requisitos: Ter uma plataforma de escrita em Python, Jupyter notebook ou executar no prompt de comando
* Instalação: Copiar e colar o código; fazer o download da base de dados, executar o código
  
* <a href="https://github.com/lucasdga1/Modelo_Regressao_logistica/blob/main/CARDIO_BASE.csv">Base de dados</a>
* <a href="https://github.com/lucasdga1/Modelo_Regressao_logistica/blob/main/Regressao_logistica_pratique.ipynb">Código do modelo</a>
